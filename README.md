# ⚡ Time Attack 國中英語單字挑戰 🚀

**👉 [點擊這裡立即開始遊戲 (Live Demo)](https://angela-godbless.github.io/-/) 👈**

## 🎯 專案簡介
這是一個基於 HTML、CSS 和 JavaScript 實現的**中翻英單字測驗遊戲**。本專案將國中英語教材單字與遊戲化 (Gamification) 概念結合，加入了「時間壓力」、「連擊加成」和「最高紀錄」等機制，幫助學生在刺激有趣的環境下快速記憶與反射性地運用單字。

## ✨ 遊戲特色

* **⏳ 時間壓力 (Time Attack):** 每題限時，倒數計時 Bar 隨著時間縮短，並在危險時變色提醒。
* **🔥 連擊加成 (Combo System):** 連續答對可獲得 Combo 加成，分數倍數成長，激勵玩家追求更高的連勝。
* **🚀 速度獎勵:** 答題越快，獲得的分數獎勵越高。
* **👑 最高紀錄:** 透過瀏覽器 `localStorage` 儲存最高得分紀錄，方便學生互相挑戰或自我超越。
* **📚 單字內容:** 使用國中英語教材 **L5 進階單字** (約 40+ 個單字)。

## 🕹️ 怎麼玩？

1.  點擊網頁上的 **[開始遊戲]** 按鈕。
2.  畫面中央會出現一個**中文單字**。
3.  在輸入框中輸入該單字或片語的**英文翻譯**。
4.  點擊 **[確認答案]** 或按下鍵盤 **[Enter]** 鍵。
5.  如果答錯，遊戲結束；如果答對，分數增加，並立即進入下一題！

## 🛠️ 如何啟動？ (本地端)
本專案為純前端網頁，無需伺服器或複雜設定：

1.  將 `index.html` 檔案儲存到您的電腦中。
2.  **雙擊** `index.html` 檔案。
3.  網頁將自動在您的瀏覽器中開啟，即可開始遊戲。
